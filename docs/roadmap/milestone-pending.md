# Milestone Pending

## Goal

未確定の要望、後続候補、実装中に見つかった改善案を一時的に置く。

## Pending Items

- 丸プロップの外周歩行対応
- ブロックプロップの外周歩行対応
- RealSense 輪郭から任意形状 `WalkableSurface` を生成する
- 小人の本番アニメーション素材
- 雨、星以外の環境リアクション
- Unity Audio から FMOD への移行検討
- 外部 master data 化
- 展示用設定画面または operator debug panel

## Promotion Rule

pending item は、次の条件を満たしたら新しい milestone または既存 milestone の work item へ昇格する。

- ゴールが1文で説明できる
- 対象範囲が決まっている
- Unity 上で確認できる acceptance check がある
- 既存 milestone の完了を妨げない

## Notes

ここは決定済み仕様ではない。実装時は必ず design docs または milestone roadmap に昇格してから作業する。
