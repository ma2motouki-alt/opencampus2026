# Milestone 9: Exhibition Readiness

## Goal

展示環境で安定して動かせる状態にする。

## Current Hardware Assumptions

- Windows PC
- Unity `6000.4.10f1`
- RealSense D435
- 52インチTVを画面上向きに設置



## Acceptance Check

- 起動後すぐワールドが表示される。
- RealSenseなしでもMouse inputで確認できる。
- RealSenseありで手や物体の輪郭がUnityに表示される。
- 小さいノイズが過剰に表示されない。
- 手や物体に対して小人・粒が反応する。
- 粒を雲に触れさせると雨が降る。
- 雨から植物が育つ。
- 10分以上動かして大きく破綻しない。
