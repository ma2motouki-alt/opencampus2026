# Milestone 9: Exhibition Readiness

## Goal

展示環境で安定して試せる状態にする。

## Target Scope

- Windows PC 実行
- 水平ディスプレイ表示
- 保護アクリル板
- RealSense 上部固定リグ
- 起動時初期化
- デバッグ表示
- パフォーマンス確認

対象外:

- 新しいゲームルール追加
- 大規模な素材差し替え
- 外部マスタ基盤導入

## Work Items

- Windows PC 上で Unity アプリを起動確認する。
- 水平ディスプレイに表示して見え方を確認する。
- 保護アクリル板越しの視認性を確認する。
- RealSense 上部固定リグ前提でキャリブレーションを確認する。
- デバッグ表示 ON/OFF を展示運用しやすくする。
- 起動時の初期化と復帰を確認する。
- 長時間実行時の破綻やパフォーマンスを確認する。

## Acceptance Check

- 起動後すぐ小人ワールドが表示される
- 入力がなくても世界が動いている
- 手やプロップを置くと反応が見える
- 10分以上動かしても大きな破綻がない
- デバッグなしの通常表示で展示物として見られる

## Handoff Notes

この milestone は機能追加ではなく展示化である。新しい要望が出た場合は pending に入れ、展示安定性を優先する。
